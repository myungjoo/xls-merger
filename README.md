# xls-merger
Excel file merger

This merges .xlsx files. (not .xls yet)

# How to install

1. copy xls-merger.ps1 to c:\xls-merger\
2. copy register-mxls-merger.reg to your PC and run it.

# How to use

## In file explorer

1. right click in the folder of to-be-merged xlsx files.
2. click "이 폴더의 Excel 파일 병합하기"

## In powershell

1. go to the directory where to-be-merged xlsx files are.
2. run xls-merge.ps1
